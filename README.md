# منصة "إنجاز" 🕌 - متابعة واجبات الطلاب

منصة بسيطة لمتابعة واجبات وإنجازات الطلاب في الحلقات والمراكز التعليمية.

## المميزات
- لوحة معلم: إضافة/تعديل/حذف الطلاب، إنشاء مهام بنقاط وتاريخ تسليم، منح نقاط إضافية مع رسائل تشجيعية، إحصائيات شاملة.
- لوحة طالب: عرض النقاط، الترتيب، نسبة الإنجاز، المستويات (مبتدئ → مجتهد → بارك الله فيه → محسن → متقن)، الستريك اليومي.
- صفحة المهام: مكتملة / حالية / قادمة، مع إمكانية رفع صورة التسليم.
- لوحة التصنيف (Podium) 🥇🥈🥉.
- أفاتارات إسلامية يختارها الطالب بنفسه.
- رسم بياني لتطور النقاط عبر الوقت.

## التقنيات
- **Next.js 14** (App Router) + TypeScript + Tailwind CSS
- **Supabase** (قاعدة بيانات + تخزين صور) - مجاني بالكامل

---

## خطوات التشغيل

### 1. إنشاء مشروع Supabase
1. اذهب إلى [supabase.com](https://supabase.com) وأنشئ مشروعاً جديداً (مجاني).
2. من **SQL Editor**، نفّذ كامل محتوى ملف `supabase_schema.sql` الموجود في هذا المشروع.
3. من **Storage**، أنشئ Bucket جديد باسم `uploads` واجعله **Public**.
4. من **Project Settings > API**، انسخ:
   - `Project URL`
   - `anon public key`

### 2. إعداد المشروع محلياً
```bash
npm install
cp .env.local.example .env.local
```
ثم افتح `.env.local` وضع قيم Supabase وكلمة مرور المعلم.

### 3. تشغيل المشروع
```bash
npm run dev
```
افتح المتصفح على: http://localhost:3000

### 4. النشر (مجاني)
يمكنك نشر المشروع مجاناً على [Vercel](https://vercel.com):
- اربط المشروع بـ GitHub وارفعه إلى Vercel.
- أضف نفس متغيرات البيئة (`.env.local`) في إعدادات المشروع على Vercel.

## ملاحظات
- المعلم يسجل دخوله بكلمة مرور واحدة فقط (محددة في `NEXT_PUBLIC_TEACHER_PASSWORD`).
- كل طالب له اسم مستخدم وكلمة مرور يضيفهما المعلم من لوحة "إدارة الطلاب".
- عند إنشاء مهمة جديدة، تُنشأ تلقائياً سجلات "قيد الانتظار" لكل الطلاب (عبر Trigger في قاعدة البيانات).
- نظام الستريك (الأيام المتتالية) جاهز في قاعدة البيانات (`streak`, `last_active_date`) — يمكن تفعيله بإضافة منطق تحديث يومي بسيط لاحقاً إن رغبت.
